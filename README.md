# Delivery Station Homebrew Tap

This tap provides a Homebrew formula for installing Delivery Station (ds).

## Installation

### Add this tap

```bash
brew tap delivery-station/ds
```

### Install Delivery Station

```bash
brew install delivery-station
```

## Usage

After installation, run:

```bash
ds --version
ds --help
```

## Updating

```bash
brew update
brew upgrade delivery-station
```

## Uninstallation

```bash
brew uninstall delivery-station
```
